Teleporter Dash

Run TDMenu.html to start the game!

Project Structure:

  • /Sound
    • /Basic Soundeffects - Contains game sound effects and menu music
    • /Level Soundtracks - Contains music for all levels
  • /Levels
    • gameloader.html - Main game engine file
    • level1.js, level2.js, etc. - Individual level data files
  • /Level Editor
    • leveleditor.html - Level creation tool

Main Files:

  • TDMenu.html - Main menu and game launcher
  • readmefirst.html - This documentation file

Notes:

  • Custom levels created in the level editor will be exported as zip files
  • To add custom music, place audio files in the /Sound/Level Soundtracks folder
  • The game automatically detects and loads all level files in the /Levels directory
